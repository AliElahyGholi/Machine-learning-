## Project Description
This project presents an online football talent scouting system using machine learning techniques.
The system combines text-based player data and pose estimation from images/videos to evaluate player potential.

## Project Stages
1. Text-based position recommendation
2. Pose estimation using MoveNet (TensorFlow Lite)
3. Player potential prediction (academy acceptance & skill scores)
